## Build & Run
```
apt install docker.io -y && apt install docker-compose -y
git clone https://github.com/1makarov/go-crud-example
cd go-crud-example/
// add .env file
docker-compose up -d --build
```
## Swagger
```
http://localhost:3001/swagger/index.html
```
